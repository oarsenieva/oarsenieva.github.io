Wecreu - (capstone project)


This web application was a group project with classmates to finish our Seneca program requirements of PRJ566 and PRJ666.
Wecreu generates an online store for a small business so they can save money on IT. It works by requiring a client company to register and select one of 4 templates. Then a script creates a new folder containing the company's new online store website. This online store website is hosted on the Wecreu server. The client company is able to input and edit the information on their site as well as manage inventory and post promotions.

Overall this project consisted of the Wecreu site with registration and account management capacity, 4 front-end templates, a database that holds company information, and multiple back-end scripts that deal with validation, database access and new website generation. This project was worked on for 8 months January to August 2017, and it was planned by 4 people and implemented by 5. Originally we intended to generate entirely customizable websites, but we would only have 4 months to develop, so we decided that 4 templates would be more manageable in these time constraints.


Environment: Apache Server, MySQLi database, Languages: PHP, HTML, CSS, JavaScript, Bootstrap Other: MyPHPadmin, Unix system Hosted on Seneca College Server.
My contribution to this project consisted of:

  • Developing frontend and backend for managing inventory and database.
  • Collecting, validated user data from forms and stored it in MySQLi database.
  • Creation of one of the templates
  • Scripts for store check out
  • UI Mock ups for templates
  • Database Design and creation
  • Use cases and planning
  • Being group leader, taking notes on meetings, timesheets
